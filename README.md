 


## Run Locally  
Clone the project  

~~~bash  
  git clone https://github.com/YeminsonSanchez/dragonBallApp
~~~

Go to the project directory  

~~~bash  
  cd dragonBall
~~~

Install dependencies  

~~~bash  
npm install
~~~

Start the server  

~~~bash  
npm run dev
~~~
